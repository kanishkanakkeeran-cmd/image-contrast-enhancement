## 🎯 Project Description
This project enhances image contrast using linear transformation. It improves image visibility by modifying pixel intensity values using contrast and brightness parameters.

---

## 🧠 Mathematical Concept

g(x, y) = α · f(x, y) + β

Where:
- f(x,y): Original image
- g(x,y): Enhanced image
- α: Contrast
- β: Brightness
